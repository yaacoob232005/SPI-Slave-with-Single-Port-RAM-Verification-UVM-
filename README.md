# SPI Slave with Single-Port RAM Verification

A complete UVM-based verification environment developed to verify an SPI Slave connected to a Single-Port RAM.

## Overview

This project implements a coverage-driven verification environment using Universal Verification Methodology (UVM) to validate SPI communication and memory transactions.

## Features

* Complete UVM Testbench Architecture
* Driver, Monitor, Agent, Sequencer, and Scoreboard
* Constrained-Random Stimulus Generation
* Functional Coverage Collection
* Assertion-Based Verification
* Golden Reference Model
* Coverage-Driven Regression Testing

## Verification Scenarios

* Write Operations
* Read Operations
* Address Decoding
* Memory Access Validation
* Protocol Compliance Checks
* Corner Cases
* Error Conditions

## Tools

* SystemVerilog
* UVM
* QuestaSim
